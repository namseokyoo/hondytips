# Honeytips 
나만의 꿀팁을 공유하자!

## Introduction
+ 글자수 제한은 50자
+ 작성된 팁은 기본적으로 24시간동안 유지된다.
+ 좋아요, 싫어요를 누르면 생명력이 10분씩 늘어나고, 줄어든다.
